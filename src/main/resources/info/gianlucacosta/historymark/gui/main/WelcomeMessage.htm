Welcome!


Navigating the map is as easy as with any other application - just employ the mouse (buttons and wheel) and/or the arrow keys.


You can also work with pins:


  • To add a new pin to the map, double click on the map
  • To edit an existing pin, left-click on it
  • To remove a pin, right-click on it

You can filter pins by creating and selecting ages using the toolbar.


Your pins and ages are automatically saved to an internal database; however, you can export / import
both ages and pins, by clicking on the related toolbar buttons.


For further information on the application, please visit its website.
