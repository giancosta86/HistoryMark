HistoryMark

Interactive Java app for historical events

Run with MoonDeploy Download zip View on GitHub

HistoryMark is a Java application enabling users to pin historical events onto a world-wide, interactive geographic map.

HistoryMark employs the Atlas library - which, in turn, is based on JXMapViewer 2 and OpenStreetMap.

HistoryMark is designed to be very simple and intuitive - but you can find further instructions in its practical wiki.
